## 11. Regression Modelling

### Overview

Once the dataset had been cleaned, the next step was to forecast one selected series:

Index Numbers ; Employee households ; All groups

This was chosen because it gives a broad measure of working household living costs and showed a clear long-term trend in earlier visualisations.

Two models were tested:

- Linear Regression: Simple baseline model for trend data
- Random Forest: Machine learning comparison model

### Feature Construction

To help forecasting, lagged values were created from previous quarters.

model_df["Quarter_Number"] = range(len(model_df))
model_df["Lag1"] = model_df[target_col].shift(1)
model_df["Lag2"] = model_df[target_col].shift(2)
model_df = model_df.dropna()

This gives the model access to recent history while keeping the setup simple.

### Train/Test Split

Because this is time-series data, rows were split in chronological order rather than random order.

split_index = int(len(model_df) * 0.8)

X_train = X.iloc[:split_index]
X_test = X.iloc[split_index:]
y_train = y.iloc[:split_index]
y_test = y.iloc[split_index:]

This avoids future values leaking into the training set.

## 12. Model Fitting

Linear Regression

linear_model = LinearRegression()
linear_model.fit(X_train, y_train)
pred_lr = linear_model.predict(X_test)

Random Forest

rf_model = RandomForestRegressor(n_estimators=200, random_state=42)

pred_rf = rf_model.predict(X_test)
